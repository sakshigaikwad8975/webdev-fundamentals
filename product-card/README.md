# Product Catalog

A responsive product catalog built using HTML and CSS. The project displays multiple product cards with images, descriptions, ratings, prices, and interactive hover effects.

## Features

- Product cards with images and descriptions
- Product ratings using Font Awesome icons
- Hover effects on cards and buttons
- Responsive layout using Flexbox
- Styled call-to-action buttons

## Technologies Used

- HTML5
- CSS3
- Font Awesome


## What I Learned

- Creating reusable card layouts
- Using Flexbox for alignment and layout
- Working with Font Awesome icons
- Adding hover effects and transitions
- Improving UI consistency with reusable styling

## Future Improvements

- Add discount badges
- Add shopping cart functionality
- Make the layout fully responsive with CSS Grid
- Filter products by category
- Add JavaScript interactions
