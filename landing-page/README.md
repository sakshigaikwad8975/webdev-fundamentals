# Startup Landing Page

A responsive startup landing page built using HTML and CSS. This project features a modern user interface with a navigation bar, hero section, feature cards, pricing section, and footer.

---

## Features

- Responsive navigation bar
- Hero section with call-to-action button
- Feature cards with icons
- Pricing plans section
- Interactive hover effects
- Responsive layout for smaller screens
- Modern and clean UI design

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- Font Awesome Icons

---

## What I Learned

- Creating structured webpage layouts
- Using Flexbox for responsive design
- Building reusable UI components
- Adding hover effects and transitions
- Improving responsive behavior using media queries
- Organizing sections in a landing page
